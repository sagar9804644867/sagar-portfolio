# Sagar Chaudhary - Performance Engineering Portfolio

A modern, interactive portfolio website showcasing 5.6 years of experience in Performance Engineering, SRE, and AI/LLM Observability.

## Features

- **Responsive Design** - Mobile-friendly across all devices
- **Modern Aesthetics** - Dark theme with gradient accents and smooth animations
- **Project Showcase** - Interactive case studies with detailed project information
- **Technical Stack** - Comprehensive skill categorization
- **Contact Integration** - Direct email, LinkedIn, and phone links

## Tech Stack

- **React 18** - UI framework
- **CSS3** - Modern styling with animations and gradients
- **Lucide React** - Icon library

## Getting Started

### Prerequisites
- Node.js 14+ and npm

### Installation

```bash
git clone https://github.com/yourusername/sagar-portfolio.git
cd sagar-portfolio
npm install
```

### Running Locally

```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view in browser.

### Building for Production

```bash
npm run build
```

## Deployment

### Deploy to Vercel (Recommended)

1. Push to GitHub
2. Connect your GitHub repo to Vercel
3. Vercel automatically deploys on every push

Or use the Vercel CLI:

```bash
npm install -g vercel
vercel
```

### Deploy to Netlify

```bash
npm run build
```

Then drag and drop the `build` folder to Netlify.
